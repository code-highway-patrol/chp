# CHP Investigation

Perform deep investigation of specific code areas for compliance and traffic violations.

## Usage

Invoke this skill when:
- User asks about specific files or directories
- User wants to understand why something was flagged
- Targeted analysis is needed beyond general health check

## Investigation Modes

1. **Traffic Stop**: Analyze specific file(s) in depth
2. **Speed Trap**: Find all occurrences of a pattern
3. **Accident Investigation**: Explain specific violation context
4. **Highway Patrol**: Track code quality changes over time

## Process

1. Identify target scope (file, pattern, or violation)
2. Gather context and related code
3. Apply relevant CHP traffic laws
4. Analyze findings with detailed explanations
5. Provide actionable recommendations

## Output

Returns detailed investigation report with:
- Target scope and context
- All applicable law checks
- Detailed findings with explanations
- Code snippets showing violations
- Specific fix recommendations
