# Marketplace

Search and install CHP laws from the community marketplace.

## When to Use

Use when you need to:
- Find existing laws for common rules
- Install law packs for specific technologies
- Discover what enforcement options are available
- Add rules without writing them from scratch

## Commands

```bash
# Search for laws
chp search "query"

# List all available laws
chp list

# Install a law
chp install <slug>
```
