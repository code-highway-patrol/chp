# === Law: no-console-log ===
intent: No console.log statements in production code. Use a proper logging library or remove debug output entirely.
reaction: block

# === Law: no-hardcoded-secrets ===
intent: No hardcoded API keys, passwords, tokens, or secrets in source code. All sensitive values must come from environment variables or a secrets manager.
reaction: block

# === Law: no-todo-comments ===
intent: No TODO, FIXME, HACK, or XXX comments left in code. Resolve them or track them in an issue tracker.
reaction: warn

# === Law: no-debug-code ===
intent: No debug code left in production. This includes debugger statements, console.debug calls, and any code that exists solely for debugging purposes.
reaction: block
